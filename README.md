# Proyecto-intermodular-S.O
Proyecto entregable de la asignatura de implantación de sistemas operativos.
En este proyecto de la asignatura de Implantaciíon de sistemas operativos, he planteado una gestion de un dominio loca de un gimnasio. 
He seleccionado dos sistemas operativos para este dominio, uno para el servidor local (windows server) donde se administraran todas las grstiones, y otro para el dominio local (windwos 10), para que los trabajadores puedan conectarse con sus credenciales, y administrar las tareas del dia a dia.

2. En segundo lugar creo un directorio activo para que el servidor gestione los usuarios del gimnasio, y paera que los trabajadores se conceten a windwos server a travaes de un escritorio remoto, con sus correspondientes crendenciales.
3. Creo varias carpetas, donde en cada una se encuentran sus respectivos departamentos y funciones.
4. A parte de estas carpetas, tabien creo varios grupos donde se encuentras los respectivos departamentos.
5. Administro los permisos de cada usuarios, donde Dirección tiene permiso de lectura, escritura y ejecución. Administracion tiene permiso de lectura y escritura, y Entrenadores tiene solo permiso de lectura.
